# Rocket-Avionics
A compact, high-performance rocket avionics system using Raspberry Pi Pico 2 with XBee 3 Pro, SIM800L, and NavIC GPS for communication and tracking. Includes BNO085, BMP390, INA219, MAX31856, DS3231 RTC, and SD card for sensing, logging, and flight control by Team Sudarshan (IN-SPACe).
